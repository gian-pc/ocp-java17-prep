# OCP Java SE 17 Prep (1Z0-829)

Estudio autodidacta y práctica diaria para la certificación **Oracle Certified Professional: Java SE 17 Developer**.

## Estructura
- `src/main/java/ocp/` → código por semana y día
- `docs/` → explicación por tema + trampas OCP + preguntas + ejercicios + soluciones
- `quizzes/` → quizzes semanales

## Rutina (L–V)
- 1–2 commits por día
- Teoría (notas) + práctica (código) + preguntas tipo examen

## Progreso
